# The Great Vanderbilt Hackathon of March, 2025

## Use the Container

 * Get the container
 * Get into a directory of your project
 * Run the container

 Example:

 ```bash
 docker pull ...
 cd ${your_project_workdir}
 docker run --hostname=vandyhack -v .:/app/homedir -it adviser-cli:latest
 ```
